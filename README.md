# PROYECTO INTERMODULAR – 2º SMR
## Implementación de Nube Privada con Apache CloudStack
### Empresa: Lucsea (Punta Umbría, Huelva)

---

## Descripción del Proyecto

El presente proyecto intermodular tiene como finalidad el diseño de una infraestructura de nube privada basada en Apache CloudStack, orientada a su posible implantación en la empresa Lucsea, negocio unipersonal dedicado a servicios informáticos y técnicos.

El sistema permitirá la gestión centralizada de máquinas virtuales y recursos informáticos, facilitando la realización de pruebas técnicas, configuraciones y copias de seguridad sin afectar a los equipos físicos de la empresa.

---

## Objetivo General

Disponer de un sistema de nube privada que permita gestionar de forma centralizada máquinas virtuales en la empresa Lucsea, simulando su funcionamiento mediante un entorno de laboratorio.

---

## Contexto Empresarial

Lucsea es una empresa unipersonal ubicada en Punta Umbría (Huelva), dedicada a la venta de productos tecnológicos y a la prestación de servicios informáticos y técnicos.

El proyecto surge de la necesidad de mejorar la organización y gestión de los recursos tecnológicos disponibles en la empresa.

---

## Tecnologías Utilizadas

- Apache CloudStack
- Entorno de virtualización mediante VirtualBox
- Sistemas operativos basados en Linux
- Git y GitHub para el control de versiones
